Add sliding captcha fields in Contact Form 7


== Description ==

Protect Contact Form 7 forms from spam entries. Requires contact form 7.

Users are required to confirm that they are not a robot before the contact form 7 can be submitted. Itâ€™s an easy for people to just drag/slide arrow from left to right to activate for to submit data and hard for bots.

= How does it work? =

After activation plugin, go to any of your **Contact Form 7** forms and add this shortcode, **[slidingcaptcha slidingcaptcha-xyz]** before submit button where xyz means any number to separate field by id.

If you have a feature suggestion or idea you'd like to see in the plugin, we'd love to hear about it! [Suggest a Feature](https://wordpress.org/support/plugin/sliding-captcha-for-contact-form-7/)


== Installation ==

1. Upload the files to the plugin folder of your site
2. Activate it from the Plugins Page
